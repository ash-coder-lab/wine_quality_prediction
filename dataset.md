COLUMN DETAILS
========================================================

1. fixed acidity
   - Measures fixed acids in wine
   - Numeric feature

2. volatile acidity
   - Measures volatile acids contributing to vinegar taste
   - Numeric feature

3. citric acid
   - Amount of citric acid in wine
   - Numeric feature

4. residual sugar
   - Remaining sugar after fermentation
   - Numeric feature

5. chlorides
   - Salt content in wine
   - Numeric feature

6. free sulfur dioxide
   - Free SO2 amount preventing oxidation
   - Numeric feature

7. total sulfur dioxide
   - Total sulfur dioxide content
   - Numeric feature

8. density
   - Density of wine
   - Numeric feature

9. pH
   - Acidity/basicity level of wine
   - Numeric feature

10. sulphates
    - Sulphate concentration
    - Numeric feature

11. alcohol
    - Alcohol percentage in wine
    - Numeric feature

    - Wine quality score
    - Target variable/output label

    - Unique identifier for each record

PROJECT OBJECTIVE
========================================================

The main objective of this project is to develop and evaluate machine learning models capable of accurately predicting wine quality based on various physicochemical properties of wine samples.

The project aims to identify which wine characteristics most strongly influence wine quality and determine the best-performing machine learning algorithm for classification.
